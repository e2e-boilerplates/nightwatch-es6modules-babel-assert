# Nightwatch Boilerplate
    
Nightwatch end-to-end test automation boilerplate, Using [ES Modules](https://hacks.mozilla.org/2018/03/es-modules-a-cartoon-deep-dive/), [Babel](https://babeljs.io) and [Commonjs](https://nodejs.org/api/assert.html).
    
## Getting Started
    	1. git clone git@github.com:e2e-boilerplate/nightwatch-es-modules-babel-assert.git
    	2. cd nightwatch-es-modules-babel-assert
    	3. npm install
    	4. npm run test
        
    
For more boilerplate click [here](https://github.com/e2e-boilerplate/utils/blob/master/docs/implemented.md)
